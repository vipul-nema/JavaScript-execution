the width define the content space. padding is defined space outside content to border, border define space outside padding/content to margin/outline, outline do not create extra space, if defined it just overlap with other. defined outside border, but not increase extra spacing. Margin define space outside border. if total space if larger than parent space, child span outside to parent.
